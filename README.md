FormComponents
==============

Initialize HTML form elements for keeping their behavior but custom styling.

## Component List:
* Text Input
* Email Input
* Tel Input
* Url Input
* Search
* Radios            (_radio.scss)
* Checkboxes        (_checkbox.scss)
* Textarea
* Selector
* File Picker
* Numeric Input
* Range Bar
* Switcher
* Buttons
